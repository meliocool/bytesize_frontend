# BYTESIZE - Frontend

Content-addressable storage with chunk-level deduplication, exposed via a REST API.  
Built with **Go**, **PostgreSQL**, and designed to scale with **goroutines**, **channels**, and (later) **Docker/K8s**.

## Tech Stack

- React with Next.Js
- TypeScript
- TailwindCSS
- ShadCN Components
- Axios for API Calls
